# Alpha2048
> Play 2048 Game with Artificial Intelligence

## Features
+ Home-made 2048 Game Engine

![](./pics/2048.png)

+ Reinforcement learning similar with Alpha Zero

## Requirements
+ Numpy
+ Keras,Tensorflow, Theano (For reinforcement learning)
+ PyQt5 (For UI)

## How to play
+ Human play:

    ```$python alpha2048/Game2048.py```
